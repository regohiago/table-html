### Descrição do Projeto

Este projeto consiste em uma página web desenvolvida em HTML puro, simulando o site institucional de uma clínica médica. O objetivo é demonstrar a aplicação prática dos principais elementos estudados, incluindo estruturação de conteúdo, formatação de textos, uso de mídias, tabelas informativas e formulários de contato, sem utilização de CSS externo ou JavaScript.

---

## README — Documentação

### 📌 Objetivo

Construir um site demonstrativo de uma clínica médica utilizando apenas HTML, aplicando os conceitos fundamentais de:

- Estrutura de página
- Formatação de texto
- Inserção de imagens
- Tabelas
- Iframe (Google Maps)
- Formulários

---

### 🏥 Tema

Clínica médica fictícia: Clínica Vida Plena

---

### 🧱 Estrutura do Projeto

O projeto foi desenvolvido em um único arquivo `index.html`, contendo múltiplas seções internas simulando páginas:

- Página Principal
- Sobre a Clínica
- Horário de Atendimento
- Contato

A navegação ocorre via âncoras internas (`#id`).

---

### 🧭 Navegação

Menu lateral contendo links para as seções internas:

- #principal
- #sobre
- #horarios
- #contato

---

Estruturação:
- header
- main
- section
- footer
- aside

Textos:
- Títulos (h1, h2, h3)

Mídias:
- Imagens (img)

- table, thead, tbody, tr, td, th

Mapa:

- form
- fieldset
- legend

- textarea

### 📊 Tabela de Horários

Tabela informativa exibindo serviços e horários de atendimento conforme o enunciado do desafio.

---

### ✉️ Formulário de Contato

Campos implementados:

- Nome (text)
- E-mail (email)
- Assunto (text)
- Mensagem (textarea)

Botões:

- Enviar (submit)
- Limpar (reset)

---

### 🎨 Estilização

A estilização foi aplicada utilizando atributos inline (`style`), sem uso de CSS externo.

---

### 🚀 Execução

Abrir o arquivo `index.html` em qualquer navegador moderno.

### 📎 Observações

Projeto demonstrativo, sem backend ou processamento real do formulário.
